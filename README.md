# 🎓 졸업 자격 자가진단 및 성적표 자동 시각화 시스템
> **VBA 기반 학사 행정 자동화 솔루션**

사용자의 성적표 데이터를 분석하여 학과/학번 졸업 요건 충족 여부를 판단하고,<br>
성적표 시트에 카테고리별 자동 채색 기능을 제공하는 Excel의 VBA 기반 자동화 도구.

---

## 1. 프로젝트 개요
- **배경**: 근로장학생 근무 중 매 학기 복잡한 졸업 요건 대조 작업을 종이문서를 대조하며 수작업으로 진행.<br> 따라서 소요되는 시간을 단축하고, 수동 확인 시 발생하는 인적 오류를 방지하고자 개발함.
- **주요 기능**: 
  - 학번/학과별 가변 졸업 기준 자동 매칭
  - 성적표 시트 내 과목 카테고리별로 상이한 컬러 자동 매핑 (전공, 교직, 교과, 실습 등)
  - 기존의 전산시스템에서 누락된 비고란 자동 기입 ("기본이수" 등) 및 폰트 최적화
  - 예외 과목(유사 과목명, 병합된 셀) 인식 로직 포함

## 2. 기술 스택
- **Language**: VBA (Visual Basic for Applications)
- **Tool**: Microsoft Excel

## 3. 주요 기능 및 로직 상세
### ✅ 지능형 과목 대조 로직 (Strict & Keyword Match)
- **전공 과목**: 성인간호학1, 2와 같이 숫자로 구분된 과목의 오인식을 방지하기 위해 '완전 일치'(문자열만 인식) 매칭 적용.<br>(동일한 과목으로 인정하는 경우의 수를 데이터 베이스에 추가하는 방법이 졸업 요건 변경 시 사용자가 주체적으로 수정할 때 편리할 것으로 예상됨)
- **교과 교육**: '국어교과교육론', '영어교과교육론' 등 학과별로 상이한 명칭을 '교과교육론' 키워드로 통합 인식하는 유연한 매칭 적용.

### ✅ 데이터 시각화 (Auto Color Mapping)
- 사용자 정의 [색상기준] 시트를 참조하여 성적표 영역(이수구분~비고)에 자동 채색.
- 각 카테고리별로 색상을 매핑하여 가독성 향상. 졸업요건에 부합하는 과목들을 이수하였는지 검토 시 용이.

### ✅ 예외 상황 처리 (Data Cleaning)
- **병합 셀 인식**: 분류명이 상단 셀에만 적혀 있는 병합된 구조(병합된 셀의 head에서만 데이터가 인식됨(A1, B1, C1의 경우 B1, C1에서 데이터를 받을 수 없다.))에서도 <br>루프문을 통해 상위 카테고리를 역추적하여 인식.
- **가변 열 대응**: 학번별로 기준표의 열 위치가 달라도 연도를 추적하여 해당 열의 데이터를 정확히 참조.
- **전공 프리패스**: 성적표 비고란에 수동으로 '기본이수'가 적힌 경우, 로직이 우선적으로 감지하여 자동 채색.(전산 시스템에서 기본이수 과정으로 인정한 과목은
